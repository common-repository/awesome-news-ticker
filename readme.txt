== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin
4. if you want to use shortcode in theme php file please place this code <?php echo do_shortcode('YOUR SHORTCODE');?> in your template

For Basic Usage.

1.General Ticker

<pre>[tickerlist id="1"]</pre>


2.Ticker From Category

<pre>[tickerlist id="2" category="1"]</pre>


3.Speed Change

<pre>[tickerlist id="3" speed="1000"]</pre>


4.Speed Change

<pre>[tickerlist id="4" typespeed="100"]</pre>


5.Heading Background Color Variation

<pre>[tickerlist id="5" heading_bg_color="#F44336"]</pre>


6.Ticker Background Color Variation

<pre>[tickerlist id="6" background="#2196F3"]</pre>


7.Text Change

<pre>[tickerlist id="7"  text="Breaking News" ]</pre>


8.Ticker Text Color  Change

<pre>[tickerlist id="8" ticker_text_color="red"]</pre>
